# Qr-generator
A simple and responsive QR Code Generator built using React.js that allows users to generate and download QR codes from text or URLs. Users can customize the QR code size, instantly preview the generated QR code, and download it as an image file.
The application uses a free external QR Code API to dynamically generate scannable QR codes, demonstrating API integration, state management, and clean UI design in React.
Built With
 -React.js
 -JavaScript (ES6)
 -CSS
 -QR Code API
